# react-cli 配置 eslint + prettier

## 简介

typescript eslint preeiter 所用插件安装在开发环境即可。

yarn add xxx --dev

## typescript 所用插件

* typescript

## eslint + prettier 所用插件

* eslint
* @typescript-eslint/parser
* @typescript-eslint/eslint-plugin
* eslint-plugin-react

* prettier
* eslint-plugin-prettier (缺少后报错 prettier module)

## 运行

启动项目

```
yarn install
```

```
yarn serve
```

项目打包

```
yarn build
```
